Приложение "Крестики-нолики"
Игра задача в которой обыграть опонента по стандартным правилам собрав вертикальную, горизонтальную или наклонную линию из одинаковых знаков, игроки ходят поочереди.
С помощью системы сборки Apache ant, проект и тесты компилируются и выполняют в определённой последовательности
Для запуска приложения необходимо ввести: ant run
